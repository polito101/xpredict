Deliver the multi-outcome event-of-binaries DATABASE SEAM as one reversible Alembic migration plus the matching ORM, with no application logic. Output: `0011` migration file; `MarketGroup` class + 2 nullable `Market` columns + bidirectional relationship in `models.py`; `MarketGroup` exported in `markets/__init__.py` and registered in `alembic/env.py`. No data written, no API, no UI.
