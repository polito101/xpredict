## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → POST /auth/* (untrusted form/JSON) | All auth bodies are untrusted input crossing into FastAPI; pydantic + validate_password enforce strength |
| User-supplied refresh-token cookie → DatabaseStrategy.read_token | Cookie value MUST be treated as opaque; only `_hash(token)` is used for lookups; raw token never compared as plaintext |
| Mailpit (dev) / Resend (prod) SMTP boundary | Verification + reset URLs leave the trust boundary; emails MUST be best-effort (Pitfall 5) — SMTP outage cannot block the auth state mutation |
