    1. **tasks.py**: Create backend/app/integrations/polymarket/tasks.py.

       - Import celery_app from app.celery_app. Import structlog, asyncio,
         redis.asyncio as aioredis, GammaClient, PolymarketAdapter,
         get_settings, Market, Outcome, OddsSnapshot, MarketSourceEnum, MarketStatus.

       - Module-level: log = structlog.get_logger(),
         LOCK_KEY = "xpredict:poll:polymarket:lock"

       - Helper acquire_poll_lock(redis: aioredis.Redis) -> bool:
         SETNX with TTL from get_settings().POLYMARKET_LOCK_TTL_SECONDS (default 25).
         Uses redis.set(LOCK_KEY, "1", nx=True, ex=ttl). Returns bool(acquired).

       - Helper release_poll_lock(redis: aioredis.Redis) -> None:
         Calls redis.delete(LOCK_KEY).

       - _run_poll_sync() async function (the actual async logic):
         1. Get Redis connection from get_settings().REDIS_URL (create aioredis.from_url).
         2. Acquire lock. If not acquired, log.info("poll_skipped", reason="lock_held") and return.
         3. Try: create GammaClient, call fetch_top_markets(limit=25).
         4. Get AsyncSession from the async sessionmaker (use the lazy session factory from
            app.db.session). Call PolymarketAdapter().sync_top25(session, raw_markets).
         5. Commit session. Log market_count.
         6. Finally: release lock, close GammaClient.
         7. Catch exceptions: log.error with sentry_sdk.capture_exception.

       - @celery_app.task(name="app.integrations.polymarket.tasks.poll_polymarket_top25")
         def poll_polymarket_top25() -> None:
         Runs asyncio.run(_run_poll_sync()). Celery tasks are sync; wrap async in asyncio.run.

       - _run_snapshot_odds() async function:
         1. Get AsyncSession.
         2. Query all OPEN markets (both HOUSE and POLYMARKET) with selectinload(outcomes).
         3. For each market, for each outcome, create OddsSnapshot(market_id, outcome_id,
            probability=outcome.current_odds).
         4. session.add_all(snapshots), commit.
         5. Log snapshot_count.

       - @celery_app.task(name="app.integrations.polymarket.tasks.snapshot_odds")
         def snapshot_odds() -> None:
         Runs asyncio.run(_run_snapshot_odds()).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Gamma API -> Celery worker | Untrusted JSON from external API enters poll task |
| Redis -> Celery worker | Lock state determines whether poll executes |
| Backend API -> Frontend | Market data with volume/source_url served to browser |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-05 | Denial of Service | poll_polymarket_top25 | mitigate | Redis SETNX lock (TTL=25s < 30s interval) prevents overlapping polls; single batch call (2 req/min) stays under 300 req/10s limit; tenacity stops after 3 attempts |
| T-06-06 | Tampering | Redis lock key | accept | Fixed key pattern "xpredict:poll:polymarket:lock"; only poll task uses it; TTL auto-expires on crash; attacker would need Redis access (internal network only) |
| T-06-07 | Information Disclosure | source_url field | mitigate | URL is derived from source_market_id (server-controlled string), not user input; no server-side fetch of the URL; rendered as external link with rel="noopener noreferrer" in frontend |
| T-06-08 | Spoofing | Market data from Gamma | mitigate | Pydantic strict parsing (Plan 01); volume as Decimal from strings; status derived by state machine; no direct storage of raw Gamma response |
| T-06-SC | Tampering | npm/pip installs | mitigate | No new packages installed in this plan; all dependencies from pyproject.toml |
</threat_model>
