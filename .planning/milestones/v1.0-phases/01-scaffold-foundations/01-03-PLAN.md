## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| docker-compose bridge | internal traffic between backend ↔ db ↔ redis ↔ mailpit ↔ frontend; plaintext (dev only) |
| host → compose | Pol's machine binds ports 5432/6379/8000/3000/5555/8025/1025; no external exposure beyond `localhost` in dev |
| app → audit_log | every audit insert MUST go through `AuditService.record()`; raw INSERT is allowed at DB level (the trigger only blocks UPDATE/DELETE), but app-layer convention forbids it per D-20 + CONVENTIONS.md (created in 01-01) |
| DB connection pool | asyncpg pool with `pool_pre_ping=True`, `pool_recycle=3600`, `pool_size=10`, `max_overflow=10` per D-41 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
